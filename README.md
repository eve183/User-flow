# User-flow
Sistema de pedidos de cafetería universitaria

Este proyecto propone una aplicación que permite a los estudiantes de la universidad realizar pedidos de comida desde su celular.
El sistema busca reducir las filas en las cafeterías, mejorar la organización de los pedidos y optimizar el tiempo de los estudiantes dentro del campus.


1. Registro de usuario
Problema

Los estudiantes no cuentan con un sistema digital para registrarse y acceder al servicio de pedidos de comida.

Solución

La aplicación permitirá registrarse con el correo institucional y una contraseña para acceder al sistema.

Historia de Usuario

Como estudiante de la universidad
Quiero registrarme en la aplicación con mi correo institucional y una contraseña
Para poder acceder al sistema y realizar pedidos de comida.

Requerimientos

Registro con correo institucional

Creación de contraseña segura

Validación de correo institucional

Almacenamiento de datos del usuario


2. Visualización del menú
Problema

Los estudiantes hacen fila sin saber qué productos están disponibles en las cafeterías.

Solución

La aplicación mostrará el menú actualizado con productos y precios.

Historia de Usuario

Como estudiante
Quiero ver el menú disponible desde la aplicación
Para decidir qué comida pedir antes de llegar.

Requerimientos

Mostrar menú actualizado

Mostrar precios

Indicar disponibilidad de productos

Mostrar puntos de venta

3. Pedido anticipado
Problema

Los estudiantes pierden tiempo haciendo largas filas para pedir comida.

Solución

La aplicación permitirá seleccionar productos del menú y realizar pedidos anticipados.

Historia de Usuario

Como estudiante con poco tiempo entre clases
Quiero seleccionar productos del menú y realizar un pedido anticipado
Para evitar las filas en la cafetería.

Requerimientos

Seleccionar productos

Agregar productos al carrito

Confirmar el pedido

4. Pago del pedido en la aplicación
Problema

Los estudiantes deben hacer fila en la caja para pagar su comida.

Solución

La aplicación permitirá realizar el pago desde el sistema antes de recoger el pedido.

Historia de Usuario

Como estudiante
Quiero pagar mi pedido desde la aplicación
Para evitar hacer fila en la caja.

Requerimientos

Mostrar total del pedido

Permitir pago digital

Confirmar pago

Registrar el pago

5. Gestión de pedidos para cafetería
Problema

El personal de cafetería no tiene una forma organizada de ver los pedidos anticipados.

Solución

El sistema permitirá visualizar los pedidos para prepararlos con anticipación.

Historia de Usuario

Como personal de cafetería
Quiero ver los pedidos realizados por los estudiantes
Para prepararlos antes de que lleguen.

Requerimientos

Visualizar pedidos

Mostrar detalles del pedido

Actualizar estado del pedido

Organizar pedidos por orden de llegada

6. Notificación de pedido listo
Problema

Los estudiantes deben esperar cerca del mostrador hasta que su comida esté lista.

Solución

La aplicación enviará una notificación cuando el pedido esté listo para recoger.

Historia de Usuario

Como estudiante que realizó un pedido anticipado
Quiero recibir una notificación cuando mi comida esté lista
Para recogerla sin esperar.

Requerimientos

Enviar notificación automática

Mostrar estado del pedido

Identificar el pedido con un código
